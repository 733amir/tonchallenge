# TON Challenge

## Dev

### Init

* `npm init vue@latest`
* `cd tonchallenge && npm install`

#### Dependencies

* `npm install ton ton-core ton-crypto`
* `npm install @orbs-network/ton-access`
* `npm install vite-plugin-node-polyfills`
* `npm install @tonconnect/sdk`

### Dev

* `npm run dev`

### Build

* `npm run lint`
* `npm run build`
